# YouTube 動画記事変換スキル 📺 ➡️ 📝

YouTube 動画を高精度で構造化された読みやすい記事に変換するための **Hermes Agent Skill** 定義です。

## 📖 概要
単純な要約ツールとは異なり、このスキルは知識抽出のための**工業グレードのパイプライン**をカプセル化しています。AIエージェントに、字幕の不一致や詳細の喪失などの一般的な落とし穴を回避させ、最終的な出力がプロフェッショナルな編集構造を維持するようにガイドします。

## 🛠️ 処理パイプライン
本スキルは3つの段階で構成されています：

1. **抽出 (Extraction)**: 
    - `yt-dlp` を使用してメタデータと `.vtt` 字幕を取得。
    - 優先順位に基づいた言語マッチングシステム（繁体字中国語 -> 簡体字中国語 -> 英語 -> 日本語 -> 等）を実装。

2. **洗浄 (Cleaning)**: 
    - WEBVTT ヘッダー、タイムスタンプ、HTML タグを削除。
    - 連続する重複行を排除し、AI生成字幕特有のノイズを除去。

3. **統合再構築 (Synthesis)**: 
    - 線形な要約ではなく、内容を「トピックブロック」として再構成するようエージェントを誘導。
    - 具体的な事例、データ、名言を保持しつつ、口語体をプロフェッショナルな書き言葉に変換。

## 📂 リポジトリ構成
- `SKILL.md`: コア定義ファイル。トリガー条件、詳細な実行ステップ、および包括的な「落とし穴 (Pitfalls)」セクションが含まれます。
- `references/`: 高度な実装のための参考ドキュメント：
    - `apple-notes-export.md`: Apple Notes へのクリーンテキストエクスポートガイド。
    - `common-failures.md`: 変換失敗時の診断ワークフロー。

## 🚀 使用方法

### 開発者向け
このスキルを Hermes Agent のスキルライブラリにインポートしてください。ユーザーが YouTube リンクを提供し、記事への変換を要求した際、エージェントは定義されたワークフローに従って自動的に実行します。

### ユーザー向け
スキルが有効になったら、エージェントに次のように指示してください：
*「この動画を構造化記事に変換して：[YouTube URL]」*
